# 🍽️ TasteFusion

TasteFusion is a modern and responsive food/restaurant website built using HTML, CSS, and JavaScript.  
The project provides an attractive UI for showcasing food items, restaurant services, menus, and customer interaction.

---

## 🚀 Features

- 🍕 Beautiful Food Menu
- 📱 Fully Responsive Design
- 🎨 Modern UI/UX
- ⚡ Smooth Navigation
- 🖼️ Food Image Showcase
- 📋 Interactive Sections
- 💻 Lightweight & Fast

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/AliyaJaved/TasteFusion.git
```

Go to the project folder:

```bash
cd TasteFusion
```

---

## ▶️ Run the Project

### Method 1: Directly Open

Open `index.html` in your browser.

---

### Method 2: Using VS Code Live Server

1. Open project in VS Code
2. Install Live Server Extension
3. Right click on `index.html`
4. Click **Open with Live Server**
